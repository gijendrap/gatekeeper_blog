Core Capabilities

Built for the paranoid
developer in all of us.

🛡️
Zero-Trust IP Vault
Every file is private by default. You explicitly whitelist what's safe to publish. Nothing sneaks out without your approval.
🔍
Deep Secret Scanning
Powered by Gitleaks — scans for 20+ API key patterns including OpenAI, AWS, Stripe, GitHub, and raw RSA key blocks.
⚡
Shell Shim Injection
Silently intercepts git push and npm publish in Bash, Zsh, and PowerShell. Set it and forget it.
🎛️
Interactive TUI
A rich terminal UI lets you visually toggle files between PUBLIC, PRIVATE, and MIXED states with real-time inheritance logic.
🚨
Auto-Remediation
Detected a private file in your commit? GateKeeper can automatically amend your git history to strip it — no manual rebasing needed.
📦
Bloat Prevention
Detects accidental node_modules, venv, and __pycache__ check-ins before they bloat your repository.
Quick Start

From zero to
protected in 3 commands.

01
Install globally
Add GateKeeper to your machine with Python's package manager. It's a single global install — works across all your projects.
bash 
$ pip install gatekeeper-cli
02
Inject the shell shims
This is a one-time setup that silently wraps your git and npm commands. Supports Bash, Zsh, and PowerShell.
bash 
$ gatekeeper install
✅ Shims injected into ~/.zshrc
Run: source ~/.zshrc to activate
03
Initialize your project
Navigate to any repository and choose your trust model. Optionally open the interactive TUI for granular file-by-file control.
bash 
$ cd my-secret-project && gatekeeper init
Analyzing context... Found 9 ignore patterns.

Evaluation Pipeline

Every push runs through
a two-stage firewall.

STAGE 0
Bloat Check
Detects accidental inclusion of massive dependency folders and offers to amend your git history automatically.
Global · Always runs
STAGE 1
IP Vault Check
Compares every outgoing file against your whitelist using a path-specificity algorithm. Blocked files are surfaced instantly.
Per-file · Whitelist driven
STAGE 2
Secret Scan
Runs Gitleaks on all whitelisted outgoing files. Scans for 20+ enterprise secret patterns without hitting memory limits.
Gitleaks · Auto-downloaded
RELEASE
Safe to Proceed
All stages passed. GateKeeper hands control back to the real git or npm process to complete the push.
✓ Cleared
